# 🏅 Olympics Data Analysis

This project performs **Exploratory Data Analysis (EDA)** on the historical Olympic Games dataset and visualizes insights about athletes, countries, sports, and medal distributions.

The analysis is implemented using **Python, Pandas, Matplotlib, Seaborn, and Plotly**, and is inspired by the CampusX Olympics Data Analysis tutorial.

---

## 🎯 Project Objective

The goal of this project is to analyze Olympic Games data and extract insights such as:

- Medal tally by country
- Overall Olympic statistics
- Athlete participation trends
- Country performance over time
- Gender participation trends
- Sports distribution across Olympic years

## 📊 Analysis Performed

The project analyzes Olympic data across multiple dimensions.

### 🥇 Medal Tally
- Total medals by country
- Gold, Silver, Bronze distribution

### 🌍 Country Analysis
- Performance of countries across Olympic years
- Medal trends over time

### 🏃 Athlete Analysis
- Age distribution of athletes
- Height vs Weight distribution
- Male vs Female participation

### 📈 Overall Olympic Statistics

- Number of Olympic editions
- Number of participating countries
- Total sports and events
- Total athletes
